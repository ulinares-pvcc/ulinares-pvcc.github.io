Networking & The Internet
Term Definition
Network Classification
  • LAN
  • MAN
  • WAN
  • Collection of computers in a single building or building complex.
  • Medium size network such a community.
  • World network links machines over great distances.
Network Models
  • Bus Network
  • Star Network
  • Machines connected to common communication line called the bus.
  • One machines serves as central focal point to which all other machines are connected.
Protocals
  • CSMA/CD Ethernet
  • CSMA/CA Wifi
  • Carrier Sense, Multiple Access with Collision Detection: The protocol that gives the right to transmit a message on a bus network - dictates that each message is broadcast to all machines on the bus.
  • Carrier Sense, Multiple Access with Collision Avoidance: The protocol designed to avoid collisions - when collisions occur, the message must be retransmitted.
Combining networks:
  • Repeater
  • Bridge
  • Switch
  • Router
  • Gateway
  • Device that passes signals between two busses using amplification without considering the meaning of the signals.
  • More complex and more efficeint repeater that checks destination address and forwards a message only when destined for the computer on other side.
  • A bridge with multiple connections.
  • Special purpose computers used for forwarding messages that provides links between networks - they maintain a forwarding table for direction of messages.
  • The point at which one network is linked to an internet.
Client/Server Model
  • P2P: Peer-To-Peer
  • Distributed System
  • Cloud Computing
  • Interprocess communication model that provides service to and receive service from each other - executed on temporary basis.
  • Software system that controls a network infrastructure that consists of software systems that execute processes on a different computer.
  • Huge pools of shared computers on the network allocated for use by clients as needed.
Internet Addressing
  • Tier 1
  • Tier 2
  • Tier 3
  • High speed, high capacity, international WANs - the backbone of the Internet.
  • Regional in scope and less powerful the tier 1.
  • Intermediary ISP that provides access to the tier 1 and tier 2 cores.
IP address
  • ICANN
  • DNS
  • TLD: Top-level domain
  • Country-code TLDs
  • Internet Corporation for Assigned Names and Numbers - nonprofit established to coordinate the Internet's operation - awards blocks of consectively numbered IP addresses.
  • Domain Name System - name servers used as Internet-wide directory system.
  • Top-Level Domains - used to reflect the domains classification.
  • Two-letter TLDs specific for each country.
Internet Protocols
  • FTP
  • HTTP
  • SMTP
  • File Transfer Protocol - application for accessing another computer from a great distance.
  • Hyper Text Transfer Protocol - a more powerful protocol to handle today's web servers and browsers.
  • Simple Mail Transfer Protocol - defines a way that two computers on the network may interact when transmitting an email message from one host to the other.
World Wide Web
  • URL
  • Client-side Activities
  • Server-side Activities
  • Uniformm Resource Locator - Unique address given to locate and retrieve documents on the web - has information need by a browser to contact the proper server.
  • Activity required by a browser or web server for a client to interact with the page or user.
  • Activity required by a browser or web server for a server to interact with the page or user.
Internet four-layer software
  • Application Layer
  • Transportation Layer
  • Network Layer
  • Link Layer
  • Software units that use clients and servers that use Internet communication to carry out their tasks.
  • Used to send and recieve messages over the Internet - divides long messages into smaller segments.
  • Decides in which direction a packet should be sent at each step through the path along the Internet.
  • Transfers the packet - deals with the communication details of particular to the network.
Internet is:
  • Packet-switching Network
  • Fault Tolerant
  • Redundant
  • The internet is a packet-switch system which digital data is sent by breaking data into packets which contain data and control information.
  • Ability of a system (network) to continue operating without interuption when components fail.
  • There is a backup for delivery when congestion or failures occur - increases reliability and helps scale to more people and devices.
Cybersecurity
  • Malware
  • Virus
  • Worm
  • Trojan Horse
  • Spyware
  • Phishing
  • DDos Attack
  • Malacious software that can be transferred to and executed on a computer.
  • Software that infects a computer by inserting itself into programs that already reside on the machine - when the host program runs, the virus executes.
  • Autonomous program that transfers itself througha network, taking up residnece in computers and forwarding copies to other computers.
  • Enters a computer system disguised as a desirable program - they are often in form of email attachment.
  • Software that collects information and activities on a computer and reports back to the instigator.
  • Obtaining information by asking for it - email and telephone cons are examples - usually perpetrator claims to be financial institution.
  • Distributed Denial of Service attack - overloading computer with messages - launched against webservers to disrupt business.
Network Classification
  • Firewall
  • Spam Filter
  • Proxy Server
  • Antivirus Software
  • Cryptography
  • Public Key Encryption
  • Software that filters traffic passing through a point on a network.
  • Variation of firewall designed to block unwanted email.
  • Software that acts as intermeidiary between a client and server with the goal of shielding the client from adverse actions of the server.
  • Software that is used to detect and remove the presence of known viruses and other infections.
  • Study of sending and receiving messages securely in the presence of adversaries.
  • Techniques where the encryption and decryption algorithms are known, and one key ios known, but the encoded message remains secure.

Chapter 6
Programming Languages
Term Definition
Machine Language A branch of artificial intelligence (AI) and computer science which focuses on the use of data and algorithms to imitate the way that humans learn, gradually improving its accuracy.
Debugging A task that often leads to errors that must be located and corrected before the job is finished.
Assembler Developed to convert mnemonic expressions into machine language instructions.
Assembly Language A mnemonic system for representing programs is collectively.
Translator A program that converts source code into object code.
Compiler A special program that translates a programming language's source code into machine code, bytecode or another programming language.
Interpreter A computer program that directly executes instructions written in a programming or scripting language, without requiring them previously to have been compiled into a machine language program.
Cross-platform software A product or system that can work across multiple types of platforms or operating environments.
Natural language Spoken languages of the world such as English, German, Spanish, and Mandarin.
Formal Language Programming languages such as Python, C++, and Java.
Machine independece A term applied to software that is not dependent on the properties of a particular machine, and can therefore be used on any machine.
High-level programming language Provide more abstractions for the programmer and make it easier for people to read and write a program. They leverage millions of lines of library code, invoke built-in operating system functions, or initiate complex interactions with networks and databases.
Object-oriented programming A style of programming characterized by the identification of classes of objects closely linked with the methods (functions) with which they are associated.
Scripting language A programming language that employs a high-level construct to interpret and execute one command at a time.
Data Types:
  1. Integer
  2. Float
  3. String
  1. Numeric data consisting of whole numbers.
  2. Numeric data that might contain values other than whole numbers stored in floating-point notation.
  3. Text data.
Primitive data type A set of basic data types from which all other data types are constructed.
Data structure A specialized format for organizing, processing, retrieving and storing data.
Python collection data structures
  1. List
  2. Tuple
  3. Set
  4. Dictionary
  1. Used to store multiple items in a single variable. Created using square brackets.
  2. Used to store multiple items in a single variable. Ordered and unchangeable. Created using round brackets.
  3. Used to store multiple items in a single variable. Unordered, unchangeable, and unindexed. Created using curly brackets.
  4. Used to store data in key:value pairs. Ordered, changeable, does not allow duplicates. Created using curly brackets.
Function A set of instructions for performing a task that can be used as an abstract tool by other program units.
Local variable A variable declared within a function.
Global variable Variables whose scopes are not restricted to a particular part of a program.
Source program A text file that contains instructions written in a high level language.
Object program A computer program translated from the equivalent source program into machine language by the compiler or assembler.
